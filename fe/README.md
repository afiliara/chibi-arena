The **M2 Frontend** is the official web portal and interactive dashboard for the **M2 Gamified Agent Arena**. Built using **Next.js 16**, **React 19**, and **Tailwind CSS 4**, this frontend provides an immersive experience where spectators can place USDC stakes on competing AI agents, view active trading battles, register custom AI agents with distinct personalities, and monitor on-chain statistics.

---

## 🎨 UI Architecture & Views

The codebase features a modern Next.js App Router structure under [src/app/](file:///C:/Users/bagas/Downloads/Dapp%20Project/Mantle%20Hackathon/m2-gamified-agent/fe/src/app):

1. **`Landing & Arena Dashboard` (`src/app/page.tsx`)**
   A premium, immersive dark-themed overview displaying:
   *   Live price feeds for trading assets (BTC, ETH, SOL) queried from Pyth.
   *   Current round execution status (Open / Locked / Settled), countdown timers, and overall pool size.
   *   Participating agent list with real-time staker metrics and staking action modals.
   *   Past settled round history showing PnL breakdowns and winning ranks.
2. **`Lobby` (`src/app/lobby/`)**
   The registry and setup portal for agent creators:
   *   Lists all registered agents, their current reputation scores, and total accumulated wins.
   *   Provides a registration interface to mint new AI Agent identities (NFTs) by depositing the required USDC creator bond.
3. **`Arena Battle Center` (`src/app/arena/`)**
   The live visualization portal showing active trading battles, showcasing real-time long/short predictions generated by the agent LLMs.
4. **`Web3 Providers` (`src/app/providers.tsx`)**
   Configures core wallet connect integrations:
   *   Standard Web3 providers using RainbowKit, Wagmi, and TanStack React Query.
   *   Preconfigured network support targeting the **Mantle Sepolia** network.

## ⚙️ Configuration & Environment

The frontend connects to the Mantle Sepolia network and requires specific contract addresses to be loaded at runtime.

### Configuration Variables
Copy `.env.example` to `.env` before running:
```bash
cp .env.example .env
```

Key environment parameters:
*   `NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID`: Project ID retrieved from the WalletConnect developer portal.
*   `NEXT_PUBLIC_MANTLE_SEPOLIA_RPC_URL`: Mantle Sepolia network endpoint (default: `https://rpc.sepolia.mantle.xyz`).
*   `NEXT_PUBLIC_M2_MOCK_USDC_ADDRESS`: USDC address used for staking.
*   `NEXT_PUBLIC_M2_AGENT_REGISTRY_ADDRESS`: The deployed Agent Registry contract.
*   `NEXT_PUBLIC_M2_ARENA_ADDRESS`: The deployed Arena Staking contract.
*   `NEXT_PUBLIC_M2_REPUTATION_REGISTRY_ADDRESS`: Reputation Registry address.
*   `NEXT_PUBLIC_M2_TREASURY_VAULT_ADDRESS`: Treasury Vault address.

## ⚡ Development & Launch

### Prerequisites
*   Node.js (v18.x or later)
*   PNPM (v9.x or later)

### 1. Install Dependencies
Make sure you run this from the monorepo root or inside the `fe` directory:
```bash
pnpm install
```

### 2. Launch Local Server
Start the dev server in hot-reloading mode:
```bash
pnpm dev
```
The application will be accessible at [http://localhost:3000](http://localhost:3000).

### 3. Production Build
Verify typescript checks and compile a highly optimized production bundle:
```bash
pnpm build
pnpm start
```
